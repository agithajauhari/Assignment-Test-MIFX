# Assignment-Test-MIFX

## Overview
This project contains automated test cases developed using Katalon Studio as part of an assignment test.

## How to Run
1. Open the project using Katalon Studio.
2. Make sure all required browsers are installed.
3. Select the desired test suite or test case.
4. Click the Run button and choose the target browser.

## Notes
- This project is created for assessment purposes.
- Test data and scenarios are kept simple.
